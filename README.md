# Free Trial Hero

Prototype variants for the Shopify free-trial hero, reviewed via `index.html`
(Desktop/Mobile toggle, one tab per variant).

**Live preview:** https://freetrialherotests.quick.shopify.io

## Variants

- **Control** — background video hero
- **Dynamic Mosaic** — tilted animated mosaic
- **Marquee behind** — horizontal marquee composition
- **World** — expanding-universe tile animation
